# Scout-Recrafted

Scout Recrafted is a non-official port of the original Scout mod for newer Minecraft versions.

This project updates Scout for **Minecraft 1.21+**, with a focus on:
- improved stability
- bug fixes
- better inventory interaction handling
- continued compatibility with newer game versions

## Important notice

This is **not the original Scout mod**.  
It is a **community port / continuation** made to keep the mod usable on modern Minecraft versions.

**All original credit belongs to the original Scout author(s).**

Original project:  (https://www.curseforge.com/minecraft/mc-mods/scout) 

Original author: (https://www.curseforge.com/members/cynosphere/projects)

## Features

- Scout backpacks and pouches updated for Minecraft 1.21+
- Improved handling for inventory-related interactions
- Fixes aimed at reducing issues in newer Minecraft versions
- Ongoing maintenance for future compatibility

## Supported version

- **Minecraft:** 1.21+
- **Mod loader:** Fabric

## Installation

1. Install the required version of **Fabric Loader**
2. Install **Fabric API**
3. Download the latest release of **Scout Recrafted**
4. Place the `.jar` file into your `mods` folder

## Dependencies

- Trinkets
- Fabric API

## Status

This project is currently maintained as an updated port for modern Minecraft versions.  
Future updates may include additional fixes and compatibility improvements.

## Disclaimer

If you are looking for the original mod, please support the original project and author.  
This repository exists to preserve compatibility and keep the mod working on newer versions of Minecraft.
